AstronoPy

AstronoPy is a NumPy-based astrophysics computation package for Python.

Just plug and play:

import numpy as np
import convert as conv
import constants as const
import module as astro

Easy, right?

Documentation

The documentation is in progress.

Constants

The file constants.py is a dictionary containing a wide variety of astrophysical constants.

Conversions

The file convert.py contains one-directional unit conversion formulae.

Formulas

module.py is a module containing astrophysical formulas that take many arguments.
