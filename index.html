AstronoPy

AstronoPy is a NumPy- and SciPy-based astrophysics computation package for Python.

Just plug and play:

import numpy as np
import scipy as sp
from astro import *

Unit conversion & dimensional analysis

qtya = Quantity(15, [msun, m], [s])
qtyb = convert(qtya, [kg, km], [hrs])
qtyb.put()

Unit simplification

qty = Quantity(15, [kg, m, s], [m, s, s])
qty.put()

Kepler's 3rd law

p = Quantity(365, [days])
a = Quantity(1, [AU])
m1 = Quantity(1, [msun])
m2 = kepler3(p, a, m1, None)
m2.put() # Earth mass calculated as a 2-body system

Distance Modulus

sunAppMag = Quantity(-26.74, [mag])
sunAbsMag = Quantity(4.83, [mag])
d = modulus(None, sunAppMag, sunAbsMag)
d.put() # Earth-Sun distance calculated from solar magnitudes

Wien's Law

y = Quantity(0.0000005, [m])
wiensLaw(y, None).put() # Sun temperature given wavelength

Escape velocity

mass = Quantity(1, [mearth])
v = escapeVel(None, mass, earthRadius)
v.put() # Earth's escape velocity
