Astronopy

Astronopy is a NumPy- and SciPy-based astrophysics computation package for Python.

Documentation

Astronopy is based on the diman Quantity class. The Quantity constructor has two parameters: value and units.

In an instance of Quantity(), value can be any pythonic number data type. However, units takes the form of a list of lists as follows:

  a_units = [[msun, 1] [m, 1], [s, -1]]

Each inner list is constructed as follows: [unit_name, exponent]. This list structure makes dimensional analysis possible.

  a = Quantity(123.45, [[msun, 1] [m, 1], [s, -1]])

For non-base SI units, such as watts or joules, use the function toBaseUnits() to create an instance of the Quantity class as follows:

  b = toBaseUnits(543.21*ten(8), 'watts')
  b.put()

Standard mathematical operators (+, -, *, /, **) cannot be used with instances of the Quantity class. Instead, use:

  • Quantity().add()
  • Quantity().subtract()
  • Quantity().multiplyBy()
  • Quantity().divideBy()
  • Quantity().power()

Units will automatically be concatenated.

Unit conversion & dimensional analysis

Like units will automatically be concatenated upon the instantiation of Quantity() and when performing the above mathematical operations. However, units of the same type (e.g. kg and msun) will not automatically be concatenated. To do so, use the convert() function.

a = Quantity(15, [[msun, 1] [kg, 1], [s, -1]])
b = convert(qtya, [[kg, 1], [kg, 1], [hrs, -1]])
b.put()

Warning: when using convert(), manually ensure that there is no dimensional mismatch.

Star Simulation & Modeling

The Star() class is still being developed. However, the theory is presented below.

Citing Katz &emdash; High Energy Astrophysics (1986), we can, for the sake of simplicity, define the steady-state equations of a star as follows:

  • \[\frac{dP(r)}{dr} = \frac{-ρ(r)GM(r)}{r^2}\]
  • \[\frac{dM(r)}{dr} = {4\pi}{r^2}{ρ(r)}\]

Astrophysics Calculations

Astronopy contains a calculator function for several common astrophysical calculations.

Kepler's 3rd Law (Newton's form)

The following example shows how to calculate the length of a year from Newton's form of Kepler's 3rd Law.

year_length = kepler3(None, Quantity(1.0, [[AU, 1]]), Quantity(1, [[msun, 1]]).add(Quantity(1, [[mearth, 1]])))
year_length.put() # Untested
